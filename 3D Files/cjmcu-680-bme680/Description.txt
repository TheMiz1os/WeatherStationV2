CJMCU-680, BME680 Ambient Temperature, Relative Humidity, Barometric Pressure, and Gas VOC Sensor

Link to CAD models:
https://grabcad.com/library/cjmcu-680-bme680-ambient-temperature-relative-humidity-barometric-pressure-and-gas-voc-sensor-1

Contains 4 sensors in 1 module.

Temperature and humidity can be used for weather monitoring, process control, etc.
Barometric pressure can be used for altitude detection for i.e. UAV, drones, hiking, etc.
Gas VOC sensor can be used for i.e. indoor air quality monitoring, or pollution detection, etc.

https://www.bosch-sensortec.com/products/environmental-sensors/gas-sensors/bme680/

Sensor specs from site:
Gas sensor			
Response time (τ 33-63%)	< 1 s (for new sensors)
Sensor-to-sensor deviation	+/- 15% +/- 15
Power consumption		< 0.1 mA in ultra-low power mode

Humidity sensor

Pressure sensor
